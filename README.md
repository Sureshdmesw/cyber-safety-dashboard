\# Integrated Automotive Cyber-Safety Compliance Dashboard (IACCD)



An integrated automotive cybersecurity and safety compliance dashboard designed to support structured compliance assessment, threat analysis, risk visibility, evidence management, and regulatory readiness across automotive stakeholders.



\## Project Overview



Modern connected and software-defined vehicles introduce increasingly complex cybersecurity risks across vehicle systems, ECUs, communication networks, software components, and external interfaces.



The Integrated Automotive Cyber-Safety Compliance Dashboard provides a centralized engineering workflow for automotive cybersecurity compliance activities.



The platform is designed around automotive cybersecurity and regulatory engineering concepts including:



\- UNECE R155

\- ISO/SAE 21434

\- Cyber Security Management System (CSMS)

\- Threat Analysis and Risk Assessment (TARA)

\- Cyber Threat Susceptibility Assessment (CTSA)

\- Cyber Risk Remediation Analysis (CRRA)

\- Automotive cybersecurity evidence management

\## Project Purpose



The purpose of this project is to explore how automotive cybersecurity compliance workflows can be digitally integrated into a centralized engineering dashboard.



The system focuses on improving visibility across cybersecurity assessment, compliance tracking, threat analysis, risk management, and evidence readiness.
